`<Description:>`

Cub3D is a graphics project inspired by the famous Wolfenstein 3D game.
The goal of this project is to explore the basics of raycasting and understand
how a 3D environment can be rendered from a 2D map using mathematics and geometry.

Through this project, you learn how rays are cast from the playerâ€™s point of view,
how walls are detected, and how distances are calculated to create the illusion
of depth. You also work with textures, colors, and basic player movement.

cub3D is an introduction to graphical programming and helps build intuition
about how game engines work at a low level, using the C language.

`<Instructions>`

#Compiling the project

1. Clone the repository.
2. Run `make` to compile the program.
3. An executable called `cub3D` will be generated.

#Running the program

1. Run the program with a map file:
   `./cub3D maps/map.cub`
2. The map file must follow the rules defined in the subject.
3. The window will open and display the 3D representation of the map.

#Controls

1. Move forward / backward using the keyboard.
2. Rotate the camera left and right.
3. Close the window using the ESC key or the window close button.

`<Submission requirements>`
